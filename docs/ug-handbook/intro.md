# 亚马逊云科技 UG China 成员手册

欢迎来到亚马逊云科技 User Group China 社区！这里是一个充满活力的开发者社区，汇聚了众多热情的技术专家和开发者。

## 🌟 社区愿景

作为一个活力满满的全球开发者社区，亚马逊云科技开发者社区中活跃着许多热情积极的社区领袖和技术专家。我们致力于为开发者提供学习、分享和成长的平台。

## 🎯 2024年目标

1. **独立运营** - 由亚马逊云科技 UG 组委会全权组织、协调和执行
2. **提升专业度** - 增加专业开发者比例至30%
3. **深度内容** - 提供更有深度的技术内容和学习资源
4. **品牌建设** - 提升 UG 在用户和合作伙伴中的知名度
5. **规模增长** - 实现用户组、用户人数及活动参与人数的增长
