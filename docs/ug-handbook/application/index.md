# 申请流程

## 🏆 Hero 申请

### 申请条件
- 在亚马逊云科技开发者社区中具有显著影响力
- 熟悉亚马逊云科技的产品和服务
- 积极参与社区分享和贡献
- 具备技术专业性和号召力

### 申请方式
- **要求**：需要内部推荐
- **申请链接**：https://dev.amazoncloud.cn/ug/form/hero
- **流程**：内部推荐 → 申请审核 → 面试评估 → 结果通知

## 👨‍💼 UG Leader 申请

### 申请条件
- 具备组织性和积极主动的特质
- 致力于建立支持性的学习空间
- 是亚马逊云科技的用户或技术爱好者
- 了解亚马逊云科技用户的行业体验
- 致力于建立包容、活跃的用户分会

### 申请流程

#### 1. 在线申请
访问 https://dev.amazoncloud.cn/community，申请成为领导者

#### 2. 填写申请表
提供以下信息：
- 个人基本信息（姓名、联系方式、职业）
- 相关技术背景
- 社区活动经验
- 申请动机和计划

#### 3. 面试评估
组委会安排面试，评估申请人的能力和适合度

#### 4. 结果通知
通过评估后，正式成为 UGL

### 面试问题示例

#### 个人背景
1. 请简述一下你自己，包括现在所在的公司、职位、技能、经验和背景
2. 在你的职业生涯中，有哪些成就是你最为骄傲的？
3. 你如何处理工作压力？有哪些应对策略？

#### 领导力和组织能力
1. 作为一名用户组领导，你认为最重要的职责是什么？
2. 如果你所在社区进行一次重要的活动，你会如何策划和准备？
3. 你认为在团队中培养领导力最重要的因素是什么？

#### 技术和社区
1. 谈谈你对亚马逊云科技有什么样的了解？谈谈你最熟悉的亚马逊云科技服务
2. 你如何吸引到更多有亚马逊云科技背景的成员参与进 UG？

## 🙋‍♀️ 志愿者申请

### 申请条件
- 有高涨的技术热情
- 有相对充裕的时间，愿意为技术社区付出
- 是亚马逊云科技的使用者与粉丝
- 拥有设计、运营、组织等技能（加分项）

### 申请方式

#### 方式一：在线注册
1. 访问 https://dev.amazoncloud.cn/ug/form/lecturerRegister
2. 点击跳转登录页面
3. 使用审核通过的账号登录
4. 完成志愿者注册

#### 方式二：活动报名
1. 关注 UGL 发布的志愿者招募公告
2. 直接联系当地活动负责人报名
3. 填写志愿者申请表
4. 参加面试或培训（如需要）

### 申请流程
1. **找到机会** - 根据活动形式找到开展志愿者注册工作的团组织负责人
2. **直接报名** - 向负责人直接报名志愿者
3. **审核评估** - 由当地活动负责人对申请人进行审核
4. **获得认证** - 审核合格并参与活动结束后，颁发注册志愿者证章

### 注意事项
- 具体的报名方式可能因活动、组织和地区而有所不同
- 建议直接联系 UGL 或查阅官方网站获取详细信息
- 完成志愿服务后可获得认证可查询的志愿者证书

## 👨‍💻 会员注册

### 注册方式
1. 访问亚马逊云科技开发者中心网站
2. 完成个人信息注册
3. 标识自己的身份（技术爱好者/专业开发者/TDM等）
4. 关注感兴趣的技术领域和活动

### 会员权益
- 参与各类技术分享和学习活动
- 使用小程序方便地注册活动
- 加入专属的技术交流群组
- 获得社区活动的优先参与权
